# GravityCompensationCode
Code required to run gravity compensation on the Exo-H3. This code was the framework used during my MEng experiment, some changes were made to facilitate the experiment and TCP/IP . This code is designed to be used on an Exo-H3.

Add the source to your path and run the scrip.
